51Nod X^2 Mod P

Posted on 2018-11-11 | In ACM 题解 | Visitors:

X*X mod P = A，其中P为质数。给出P和A，求<=P的所有X。

Input
两个数P A，中间用空格隔开。(1 <= A < P <= 1000000, P为质数)
Output
输出符合条件的X，且0 <= X <= P，如果有多个，按照升序排列，中间用空格隔开。
如果没有符合条件的X，输出：No Solution Sample Input
13 3
Sample Output
4 9

解题思路

数据较大会爆int，将变量设为long long即可
